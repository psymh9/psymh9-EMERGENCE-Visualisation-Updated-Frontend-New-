The network visualisation tool is simple and easy to use and has a few main features:

Legend: Can be referenced to find the unique colour coding for each specialty.

Node/Member Hovering: Hover over each node to reveal further information about the
member represented, i.e. name, institution, job title, specialty, location, and research
theme of interest.

Node two-factor Filtering: Allows the user to filter by two different categories
for example finding a person who specialises in Computer Science from the University of Nottingham

Node Physics: Each node on the graph is fully interactive, you can drag the nodes with
your mouse, if the node is connected to another with an edge both nodes will move.

Steps to filter nodes:

1. Under the 'Select the first filtration category' dropdown select the first category you would like to filter by i.e. Discipline, Institution etc.

2. Under the 'Select the first filtration parameter' dropdown select the parameter to filter by based on the category selected i.e. Discipline -> Computer Science

3. Under the 'Select the second filtration category' dropdown select the second category you would like to filter by i.e Discipline, Institution etc.

4. Under the 'Select the second filtration category' dropdown select the parameter to filter by based on the category selected i.e. Discipline -> Computer Science

5. Press the 'Filter' button to apply the selected filtration categories and parameters to the visualisation.

6. Press the 'Reset Selection' button to the right of the 'Filter' button to undo the filter.
